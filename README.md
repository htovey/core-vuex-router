# core-vuex-router

This project is an experiment to see how working with Vue compares to working with React, using a subset of components from a larger React project.
Likes:  Vuetify is easy to use and in some ways is less opinionated than MUI which is nice.
        Lots of other tools and frameworks are available and easy to implement in conjunction with Vue - and require less hokey pokey than React
        Decent documentation and adoption level
Dislikes: I still can get used to the convention of having the rendered elements at the top of the code.  I don't know if it's the java developer in me
          Mix of concerns-  I feel like there is more overlap between what is rendered and what is programmed-  it's a fuzzy impression but I feel like React is better at compartmentalizing what happens where.
